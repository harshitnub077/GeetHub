# Contributing to Geethub 🎸

Thanks for your interest in improving Geethub!

## Getting Started

1. Fork the repo and clone it locally.
2. Run `npm install` then `npm run dev`.
3. Configure your local database with `npm run setup-db`.

## Adding Chord Sheets

- Submit songs via the `/commit` page in the app UI.
- Use `[Chord]` notation within lyrics.

## Code Style

- TypeScript + React functional components
- CSS via CSS variables (see `globals.css`)
- No external UI libraries

## Pull Requests

- Open a PR from a feature branch
- One feature or fix per PR
- Include a brief description of the change

## Questions?

Open a GitHub issue or email geethub@dev.io
