# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅ Yes    |
| < 1.0   | ❌ No     |

## Reporting a Vulnerability

Email security@geethub.dev with:
- A description of the vulnerability
- Steps to reproduce
- Impact assessment

We respond within 48 hours and patch within 7 days of confirmation.
